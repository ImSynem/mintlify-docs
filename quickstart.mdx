## Create an Account

Visit the Payvra website and sign up for an account. Fill in the required information and complete the registration process. Once your account is created, you will have access to the Payvra dashboard.

## Generate an API Key

To start using the Payvra API, you need to generate an API key.

1. Log in to your Payvra dashboard.
2. Navigate to the Settings > Developer section.
3. Click on the "Create API Key" button.
4. Enter a name and expiration date for your API key.
